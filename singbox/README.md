## Sing-Box

### Before running
The default `server.json`/`client.json` examples include a simple Vmess-TCP configuration. It's possible to change the server's vmess inbound and client's vmess outbound to any desired protocol.

To use the default Vmess-TCP, the placeholder for values that need to change is `CHANGEME`.
Before running search for `CHANGEME` in the files and replace them with actual values.
