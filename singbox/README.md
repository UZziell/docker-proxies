## Sing-Box

### Before running
The default `server.json.example`/`client.json.example` examples include a simple Vmess-TCP configuration. It's possible to change the server's vmess inbound and the respective client's vmess outbound to any desired protocol.

To use the default Vmess-TCP, the placeholder for values that need to change is `CHANGEME`.
Before running search for `CHANGEME` in the example files and replace them with actual values; finally save them to 'server.json/client.json' files.
